# 🧠 Главные идеи тегов (MVP)

### Архитектура тегов (3 уровня)
```
Category → Dimension → Tag
```

#### Уровень 1: Category (что это за сфера)
Это “смысл” привычки:
- здоровье
- обучение
- продуктивность
- личная жизнь
- ментальное состояние

👋 Это используется для high-level аналитики

#### Уровень 2: Dimension (тип анализа)

Это ключевая штука, которую часто забывают. Dimension отвечает на вопрос:
```
“по какому признаку мы хотим анализировать?”
```

Основные измерения:
1. Тип активности
- физическая
- когнитивная
- социальная
2. Контекст времени
- утро
- день
- вечер
3. Локация
- дома
- вне дома
4. Сложность
- легкая
- средняя
- сложная
5. Тип выполнения
- бинарная (сделал / нет)
- количественная (минуты, повторения)

#### Уровень 3: Tag (конкретика)

Это уже конкретные вещи:
- бег
- чтение
- медитация
- кодинг

#### Что мы сможем считать

1. Анализ по категориям
> “Ты больше всего времени тратишь на обучение”
2. Анализ по времени суток
> “Утренние привычки выполняются на 80%, вечерние — на 40%”
3. Анализ по сложности
> “Сложные привычки чаще пропускаются”
4. Контекст
> “Дома ты продуктивнее”
5. По конкретным тегам
> “Чтение выполняется стабильно, а спорт — нет”

#### Связь с маскотом
Маскот может говорить:
- “Ты сегодня не сделал ни одной физической активности”
- “Ты держишь серию в обучении 🔥”

# 🧩 Система тегов (MVP)

## 1. Category (уровень смысла)

categories:
  - здоровье
  - обучение
  - продуктивность
  - ментальное состояние
  - личная жизнь

## 2. Dimensions (фиксированные измерения)

Это не просто теги — это оси аналитики.

Время суток (time_of_day)
  - утро
  - день
  - вечер

Тип активности (activity_type)
  - физическая
  - когнитивная
  - социальная

Контекст (context)
  - дома
  - вне дома

Сложность (difficulty)
  - легкая
  - средняя
  - сложная

Тип трекинга (tracking_type)
  - бинарная
  - количественная

## 3. Core Tags (конкретные привычки)
👉 даем ограниченный, но универсальный набор

🟢 Здоровье - health:
  - бег
  - тренировка
  - растяжка
  - прогулка
  - сон
  - питание

🔵 Обучение - learning:
  - чтение
  - изучение языка
  - онлайн-курс
  - кодинг

🟣 Продуктивность - productivity:
  - планирование
  - deep work
  - работа
  - разбор почты

🧘 Ментальное состояние - mental:
  - медитация
  - дневник
  - рефлексия
  - отдых

🧡 Личная жизнь - life:
  - семья
  - друзья
  - хобби

## 4. Как это использовать (пример)

Привычка:
```
“Читать 30 минут вечером”
```

Она "превращается" в: **category: обучение**  
tags:
  - чтение
  - когнитивная
  - вечер
  - легкая
  - дома

## 5. Что это сразу дает
Пример аналитики:

По времени суток
> вечер — 40%, утро — 80%

По типу активности
> когнитивные привычки выполняются лучше

По сложности
> сложные привычки чаще пропускаются

По тегам
> чтение стабильно, тренировки — нет

## 6. Ограничения (важно для MVP)

🚫 Ограничиваем:
- максимум 5–6 тегов на привычку
- пользователь НЕ создает свои теги

✅ Разрешаем:
- выбирать из списка
- фильтровать по тегам

## 7. Связь с маскотом

Теперь маскот может говорить осмысленно:
- “Ты сегодня не сделал ни одной физической активности”
- “Ты держишь серию в обучении 🔥”
- “Вечер — твоя слабая зона 👀”

## 8. Минимальный набор для реализации (если совсем резать)
MUST HAVE:
- категории (5)
- время суток (3)
- тип активности (3)
- 10–12 core тегов

У вас теперь:
- структурированная система
- ограниченный набор
- готовность к аналитике
- база для геймификации
